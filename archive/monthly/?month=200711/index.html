using System;
using System.Collections.Generic;
using System.Text;

namespace cw_1
{
    class Program
    {
        static void Main(string[] args)
        {
            int ile = Convert.ToInt32(args[0]);

            int[,] tab = new int[ile, ile];

            init(tab, ile);
            disp(tab, ile);

            Console.ReadKey();
        }
        static void disp(int[,] tab, int ile)
        {
            int kol = 0;
            while (kol < ile)
            {
                for (int licz = 0; licz < ile; licz++)
                {
                    Console.Write("{0,5} ", tab[licz, kol]);
                }
                Console.WriteLine();
                kol++;
            }
            Console.WriteLine();
        }

        static void init(int[,] tab, int ile)
        {
            int p, k; 
            int wiersz = 0;
            int kolumna = 0;
            int licz = 1;


            p = 0;
            k = 1;
            for (int ii = 0; ii < ile; ii++)
            {

                for (int l = p; l < ile - k; l++)
                {
                    tab[l, kolumna+p] = licz;
                    licz++;
                }
                for (int i = p; i < ile - k; i++)
                {
                    tab[wiersz + (ile - k), i] = licz;
                    licz++;
                }
                for (int licz2 = (ile - k); licz2 > p; licz2--)
                {
                    tab[licz2, kolumna + (ile - k)] = licz;
                    licz++;
                }

                for (int i3 = (ile - k); i3 > p; i3--)
                {
                    tab[wiersz+p, i3] = licz;
                    licz++;
                }

                p++;
                k++;
            }
            if ((ile%2)!=0)
                tab[ile/2, ile/2]=licz;                            
        }
    }
}

No zapal, proszę, zapal

Tuesday, 27. November 2007, 12:31:03

Megi, Oboczności

Do zimy Roch przygotował się skrupulatnie. Włożył łopatę do bagażnika, nasmarował uszczelki żeby drzwi nie przymarzły, zmienił opony na zimowe, włożył gumowe dywaniki do środka, kupił odmrażacz do zamków i szyb. Roch uważał, że do zimy jest przygotowany doskonale, że w trasie nic go nie zaskoczy. Okazało się, że prawa Murphy'ego działają.

Rano Roch wsiadł do Megi, odpalił ją i pojechał do miasta na zakupy. Postawił samochód na parkingu i rzucił się w szał zakupów. Po powrocie znowu próbował odpalić Megi.

- Brrrrrr łłłuuuuuoooo - I zdechło.
- No dlaczego grymasisz kochanie? - Zapytał Roch.
- Brrrrrr łłłuuuuuoooo - Uparta bestia.
- No zapal, proszę, zapal - Błagał Roch.

Nic z tego. Megi nie dawała oznak życia. Doskonale przygotowany do zimy Roch nie sprawdził naładowania akumulatora, a ten pozwolił się rozładować w najmniej oczekiwanym momencie. Pchanie odpada, bo Megi ma katalizator, holowanie również bo Roch nie ma linki. Pozostał telefon do Przyjaciela.

- Cześć, jesteś może w TG? - Zapytał Koyocika.
- Nie, w pracy jestem. A co? - Odpowiedział Przyjaciel.
- Aaa bo samochód mi się rozkraczył i prądu chciałem pożyczyć - Powiedział Roch.
- To mogę podjechać - Powiedział Koyocik.
- Nie, póki co spróbuje coś na miejscu zdziałać - Odpowiedział Roch.

Jak tu pożyczyć prądu, jak doskonale przygotowany do zimy Roch nie ma kabli rozruchowych? Trzeba kupić. Nie ma nigdzie. Po drodze Roch spotkał znajomy samochód z PWiK. Zatrzymał go i zapytał o możliwość pożyczenia prądu.

- No nie ma sprawy, ale tak za 30 min bo teraz jedziemy na Bobrowniki - Odpowiedzieli.

Gdy przyjechali okazało się, że nikt nie ma kabli rozruchowych. W końcu do Rocha podszedł nieznajomy starszy pan.

- Co? Rozładowany? - Zapytał.
- Tak, i w dodatku nikt nie ma kabli - Smucił się Roch.
- Ja mam - Odpowiedział starszy Pan.
- A pożyczy Pan? - Zapytał z nadzieją Roch.
- Oczywiście, już przynoszę - Odpowiedział.

Po podłączeniu do samochodu PWiK, Megi wystartowała za pierwszym przekręceniem. Roch cały w skowronkach, że po dwóch godzinach stania, w końcu może pojechać.. kupić nowy akumulator. Podjechał do sklepu z akumulatorami, pierw spoglądając do protfela ile ma funduszy, i kupił nowy akumulator i kable rozruchowe.

Roch schudł 300zł, ale ma nowy, wypasiony, akumulator i kable rozruchowe równie wypasione.

Nauka na przyszłość. Lepiej zapomnieć kupić odmrażacz do szyb, z 6zł niż zapomnieć sprawdzić akumulator i stracić 300zł, ale uczymy się na błędach. Tylko szkoda, że te błędy kosztują.

Pozdrawiam mojego Przyjaciela Koyocika, owego Starszego Pana i ludzi z firmy PWiK, którzy bez gadania mi pomogli.

Pisałem już w tej sprawie do Mikołaja, który chyba zapodział mój list gdzieś w czeluściach swojej skrzynki pocztowej lub filtr antyspamowy zablokował mój list. Piszę więc do Ciebie, kochana Gwiazdko, z prośbą o spełnienie kilku moich życzeń.

Po tym krótkim wstępie pozwolę sobie przybliżyć Ci moje tegoroczne osiągnięcia i dokonania. Częsciowo uczyniłem to w liście do Świętego Mikołaja, ale przez szacunek do Ciebie rozszerzę mój wywód, tak abyś miała jasność, że zasłużyłem na podarki.

Otóż, kochana Gwiazdko, w tym roku byłem niezmiernie grzeczny, kulturalny, punktualny i wszytko inne kończące się na ..alny, może z wyłączeniem analny, ale to kwestia, której nie będę poruszać. Do tego wykazywałem się doskonałymi wynikami w nauce, o czym możesz się przekonać, zaglądając do mojego indeksu. Gdybyś go szukała to znajduje się on w moim plecaku (tym niebieskim). Dwa szalone psy i biedny Roch

Wednesday, 21. November 2007, 16:40:15

Pniowiec, Nakło Śląskie, Świerklaniec

Czy ktoś może wytłumaczyć Rochowi w jakim celu psy gonią rowerzystów? Nie mogą gonić kierowców tirów, ciągników lub pieszych? Zawsze rowerzyści. Jeśli gdzieś wybuchnie bomba to winni są Arabowie, Masoni, Żydzi lub Cykliści. Roch zagalopował się. Od początku.

Okolice 1200. Telefon dzwoni.

- Słucham - Odezwał się Roch.
- No cześć, możemy się spotkać? Podrzuciłbym Ci indeks - Odezwał się Łukasz.
- No pewnie, że możemy - Roch i tak nic nie robił.
- To gdzie mam podjechać? - Zapytał.
- Nie, fajna pogoda jest to podjadę rowerem do Piekar. Spotkamy się koło kopca - Zaproponował Roch.
- No spoko - Powiedział Łukasz.
- Tyyy albo przed kopcem na tym skrzyżowaniu bo nie chce mi się wjeżdżać pod górę - Roch przypomniał, że kopiec to ściana płaczu.
- heheh ok - Roześmiał się Łukasz.

W ten sposób Roch miał zaplanowany wypad rowerowy. Za raz po obiedzie, co było gigantycznym błędem, Roch wyruszył w kierunki Piekar Śląskich. Pogoda cudowna, niebo bezchmurne, zacna temperatura. Przyjemność jazdy psuje ta cholerna kolka, która dopadła Rocha. Jednak już w Nakle Śląskim kolka dała za wygraną. Jednak nadeszła inna plaga. Psy. Sztuk dwie, małe, kundle jakieś ruszyły w pogoń za Rochem. Dobrze, że było z górki dzięki temu Roch nie musiał butować wściekłych kundli. Prędkość na zjeździe to całe 60km/h.

Później już tylko Świerklaniec, w prawo na Kozłową Górę i cel osiągnięty. W Piekarach czekał już Łukasz i jego indeks.

- Załatwisz mi wpis u dr P? - Zapytał.
- No pewnie, dr P mnie lubi - Odpowiedział Roch.

Bo jak nie lubić studenta, który zaliczał dr P dwa razy? Później jeszcze pogawędka i można wracać. Powrót nastąpił przez Radzionków. Jednak Roch nie byłby sobą, gdyby nie pojechał na Pniowiec, oczywiście ulicą na literkę G.

Kilometrów wyszło 50, dzięki temu do 8000 brakuje 350km.

Roch pozdrawia Łukasza i mieszkańców ulicy na G.
