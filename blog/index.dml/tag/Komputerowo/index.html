Opłacało się

Wednesday, 28. November 2007, 15:55:21

Oboczności, Komputerowo

Opłacało się wstać o 0630, ubrać się i pojechać na uczelnię. Opłacało się dlatego, że Roch mógł popisać się swoimi, skromnymi, umiejętnościami. Otóż dziś na Programowaniu Obiektowym Roch błyszczał jak mucha w ortalionowym płaszczu. Zadaniem było napisać program, który wypełni tablicę od 1 do n w sposób spiralny, czyli tak:
1 2 3
8 9 4
7 6 5

No więc Roch zabrał się do roboty i po dłuższej chwili były już pierwsze efekty. Program nawet działał i wypisywał pierwsze liczby.

- Ma już ktoś - Zapytał Dr W.
- Ja prawie mam - Odpowiedział Roch.
- Zobaczmy - Dr W wykazał zainteresowanie.
- Na razie tyle mam, ale walczę dalej - Powiedział Roch.
- No niezłe - Odpowiedział Dr W.

Po kolejnej, dłuższej, chwili Roch miał prawie skończony program, ale jedna liczba psuła cały efekt końcowy. W końcu Roch, wspólnie z JackiemM, doszli do tego. Podpowiedź Dr W okazała się również pomocna bo licz - 2 to nie to samo co licz - 1.

- Skończyłem - Powiedział Roch.
- A działa - Zapytał Dr W.
- Teoretycznie - Odpowiedział Roch.
- Zobaczmy - Powiedział Dr W.
- No ale tutaj jest puste miejsce - Zauważył Dr W.
- Fakt, nie potrafię tego rozgryźć - Smucił się Roch.
- Hmmm można to zrobić tak - Podpowiedział Dr W.
- Oo nie wpadłbym na to - Zdziwił się Roch.
- Jak Pan się nazywa? - Zapytał Dr W.
- Roch Brada - Odpowiedział Roch.
- No to ma Pan 5 - Poinformował Dr W.

I tym sposobem Roch zarobił całe 5.0. Ot i jest czym się chwalić.
Gdyby ktoś potrzebował to można sobie poczytać:
using System;
using System.Collections.Generic;
using System.Text;

namespace cw_1
{
    class Program
    {
        static void Main(string[] args)
        {
            int ile = Convert.ToInt32(args[0]);

            int[,] tab = new int[ile, ile];

            init(tab, ile);
            disp(tab, ile);

            Console.ReadKey();
        }
        static void disp(int[,] tab, int ile)
        {
            int kol = 0;
            while (kol < ile)
            {
                for (int licz = 0; licz < ile; licz++)
                {
                    Console.Write("{0,5} ", tab[licz, kol]);
                }
                Console.WriteLine();
                kol++;
            }
            Console.WriteLine();
        }

        static void init(int[,] tab, int ile)
        {
            int p, k; 
            int wiersz = 0;
            int kolumna = 0;
            int licz = 1;


            p = 0;
            k = 1;
            for (int ii = 0; ii < ile; ii++)
            {

                for (int l = p; l < ile - k; l++)
                {
                    tab[l, kolumna+p] = licz;
                    licz++;
                }
                for (int i = p; i < ile - k; i++)
                {
                    tab[wiersz + (ile - k), i] = licz;
                    licz++;
                }
                for (int licz2 = (ile - k); licz2 > p; licz2--)
                {
                    tab[licz2, kolumna + (ile - k)] = licz;
                    licz++;
                }

                for (int i3 = (ile - k); i3 > p; i3--)
                {
                    tab[wiersz+p, i3] = licz;
                    licz++;
                }

                p++;
                k++;
            }
            if ((ile%2)!=0)
                tab[ile/2, ile/2]=licz;                            
        }
    }
}

Stażystki, stażystki

Monday, 15. October 2007, 16:41:25

Komputerowo

Dziś, wczesnym rankiem, Roch poszedł do firmy, w której kiedyś odbywał praktykę i tak się zżył z tam pracującymi, że nawiedza ich prawie codziennie. I dziś nastał taki moment nawiedzenia. Przyszedł około 1000.

- Oooo dobrze, że jesteś - Cieszyli się Tomek i Krzysiu.
- Linuks nam się popsuł - Poinformowali.

Roch, ogólnie znany z tego, że jak czegoś nie popsuje to i nie naprawi zasiadł do klawiatury Pingwina i zaczął chaotycznie naciskać klawisze. Tomek i Krzysiu zgromadzili się obok Roch i bacznie obserwowali poczynania Rocha.

- Hmmm jeszcze tego spróbuje - Fuknął Roch.

Po fuknięciu nastąpił ten moment, w którym Roch wklepał reboot i walnął w Enter. System się uruchomił ponownie i nadal nie działało.

- Poddaję się - Roch poległ w walce z Pingwinem.

Krzysiu zadzwonił, gdzie trzeba i proces naprawy przyśpieszył. W czasie naprawiania się Linuksa Roch przypomniał sobie o uroczej praktykantce, która przyszła po dwie dyskietki.

- A jak tam praktykantka? - Zapytał Roch.
- Tyyyy na laborze trzeba wymienić CD-ROM - Przypomniał sobie Tomek.

Wiadomo - największe nagromadzenie stażystek / praktykantek na metr kwadratowy jest na laborze.

Roch wcielił się w praktykanta, skulił się, szedł kilka metrów za Tomkiem, niósł dwa CD-ROM'y, śrubkręty - encyklopedyczny przykład praktykanta.

- Uważaj, prezeska idzie - Ostrzegł Tomek.
- Dzień dobry - Ukłonili się.

Znaleźli się przed laborem, zadzwonili, drzwi zostały otwarte. Roch rozglądał się za stażystkami, ale najpierw trzeba dokonać wymiany CD-ROM'u. Gdy, oczywiście, Roch skończył grzebanie w komputerze weszli, razem z Tomkiem, do pomieszczenia, w którym siedziały stażystki. Sztuk cztery.

- Łooooooooooooooooooo - Wydukał Roch.
- Kopia jest z lipca - Tomek próbował odciągnąć zaślinionego Rocha od stażystek.

Teraz Roch żałuje, że wybrał informatykę, bo nie ma takich stażystek.

Gdyby jakaś stażystka potrzebowała korepetycji z informatyki lub francuskiego to, Roch podaje kontakt do siebie: jacfef@o2.pl.

Roch pozdrawia Wszystkich.

P.S: Zastanawiacie się, czy Roch był na rowerze? Otóż był i zrobił 55km, dzięki czemu ogólnie jest 7150km.

WiFi, kurna, WiFi

Monday, 1. October 2007, 08:21:07

Oboczności, Komputerowo

- Dryyyyyynnnn - Telefon wrzeszczy.
- Eeeeeeeeee - Roch się przebudził.
- Witam, z tej strony Klient, może Pan przyjechać? - Pani Ola w imieniu Klienta budzi Rocha.
- No mogę. Tylko się ogarnę - Roch musi się dobudzić.

W okolicach 0900 Roch siedział już w Megi i pędził do Miasteczka Śląskiego. Na miejscu przywitał go w progu Szef.

- Witam, wszystko się pierd***li - Szef nie owijał w bawełnę.
- A co dokładnie? - Dopytywał Roch.
- No nie ma Internetu, komputer się nie włącza i trza poprawić stronę - Wyliczał Szef.
- Ma Pan tyle czasu ile potrzeba, kasa nie gra roli - Nęcił Szef.

No więc zanęcony Roch spojrzał na komputer, potem na modem i znowu na komputer. Nic się nie świeci, a więc problem sprzętowy. W końcu jakieś wyzwanie dla Rocha. Jednak pod modemem stał UPS (zasilacz awaryjny) i też nie świecił, znaczy nie działa. Roch nacisnął przycisk na przodzie UPSa i cud! UPS zapiszczał, zazgrzytał i wszystko ożyło. Modem zaczął mrugać, a więc działa.

Roch włączył komputer, po załadowaniu systemu okazało się, że jest Internet, że wszystko działa.

- Kurna, głupio tak powiedzieć, że wszystko działa po 5 minutach siedzenia pod biurkiem - Myślał Roch.
- A sprawdzę sobie pocztę, zajrzę na bloga, posiedzę w Szefowskim fotelu - Wykoncypował Roch.

Roch, w szefowskim fotelu, czuł się jak szef, szczególnie, że w pomieszczeniu obok siedziały dwie atrakcyjne sekretarki. Już Roch chciał je wołać, tak jak to ma miejsce na filmach z cyklu dużo akcji, mało dialogów, ale pomyślał, że dwóm może nie dać rady lub szef wejdzie niepodziewanie.

No więc Roch skupił się na poczcie, aż Pani Ola nie weszła się zapytać jak tam naprawa.

- I jak Ci idzie? - Zapytała.
- No już gotowe - Roch odpowiedział minimalizując okno z pocztą.
- Juuuuuż???? - Pani Ola wyraźnie zdziwiona.
- No bo ja to szybki jestem - Roch kokietował ją.
- Ooooo to może kawki się napijesz? - Chyba jej się podobało.
- Chce Pani, żeby mi stanął... serce znaczy się - Dobry bajer to podstawa.

Chwile jeszcze pogadali i Roch zakomunikował, że wszystko gotowe i może się zbierać, Internet działa, komputer i modem też.

- Ale szefa nie ma to się potem rozliczycie - Pani Ola powiedziała na pożegnanie.
- Noł problem - Odpowiedział Roch.

Nawet to i lepiej, bo procent urośnie, a i będzie można powiedzieć, że siedziało się 3 godziny pod biurkiem.
Dobra ściema nie jest zła.

Roch zakończył swoje boje z modemem WiFi. Wynik.. 14:8 dla Rocha.

Co by tu robić?

Tuesday, 31. July 2007, 09:40:05

Oboczności, Komputerowo

Od rana Roch zadaje sobie to, jakże filozoficzne, pytanie. Odpowiedź na nie jest trudna, wręcz niemożliwa. Pogoda raczej domowa, chmurno, wietrzno, deszczowo, zimno. Rower więc pozostanie w domu, Roch też. Z jednej strony to dobrze bo Rochowe kolano będzie mogło odpocząć i przestać boleć, ale z drugiej strony trochę smutno siedzieć w domu cały dzień.

Roch dokończyłby tworzenie Rowerownika, gdyby nie złośliwość oprogramowania, które odmówiło współpracy i wypięło się swoim kompilatorem na Rocha, ale Roch należy do twardzieli i nie pozwoli sobie dmuchać w debugger. Kiedy, pomiędzy jednym drapaniem się po brzuchu, a drugim, Roch znajdzie chwilę czasu zabierze się za wrednego Borlanda i doprowadzi go do stanu używalności.

W planach jest także wypróbowanie nowego dziecka firmy Borland, czyli C++ Builder 2007, z tego co Roch czytał, znacznie poprawili zgodność z ANSI C++, można tworzyć aplikacje dostosowane (wizualnie) do Windows Vista. Roch jest ciekawy nowego środowiska programistycznego. Swego czasu korzystał z Borland C++ Builder 2006 (Pakiet Borland Developer Studio), ale jakoś nie polubił tegoż pakietu.

I w ten sposób wyszło komputerowo-programistycznie, za co Roch przeprasza.

Komputerowo, Oboczności

I o mało co nie zastrzelił tego co to cudo wynalazł. Uwaga, komputerowo będzie.

- Drrryyyyyyńńńńń - Telefon dzwoni.
- Teraz prowadzę. Niech się nagra na pocztę - Roch za kółkiem nie odbiera telefonów.
- Drrryyyyyyńńńńń - Telefon dzwoni.
- Uparty albo jakaś zakochana fanka - Wylicza Roch.
- Drrryyyyyyńńńńń - Telefon dzwoni.

W końcu Roch nie wytrzymał i zjechał na pobocze.

- Słucham! - Grzecznie powiedział.
- Witam, z tej strony Klient. Mamy problem z siecią. Może pan przyjechać? - Rochowi odechciało się żyć.
- Będę do 30min - Zrezygnowany Roch odpowiedział.

Roch wrócił do domu, przebrał się, zabrał swój komputerek, kilka RJ-45, zaciskarkę (pożyczoną) i ruszył na podbój Miasteczka Śląskiego.

- No jestem. Co się dzieje - Dopytywał Roch.
- Internetu nie ma - Dokładnie opisał problem Klient.
- SOA#1 - Pomyślał Roch.

- O motyla noga! Mam przegwizdane! - Podskoczył tak, że głowa spotkała się z blatem biurka.
